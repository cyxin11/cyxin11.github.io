DASCTF群友题web

web2

拿到题目

出现报错，存在sqlite注入

同时发现con.enable_load_extension(True)明白可以load_extension加载动态链接库getshell。且测试后可发现load_extension对后缀没有限制

然后放到ubuntu里面进行编译，编译前先安装sqlite环境：

sudo apt install libsqlite3-dev

然后编译

gcc -g -fPIC -shared exp.c -o xin.so

上传得到getshell
